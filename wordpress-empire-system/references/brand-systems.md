# Brand Systems Reference

Complete brand guidelines for Nick's 17-site publishing empire. **NEVER mix brand voices between sites.**

## Brand Voice Philosophy

Each site serves a distinct audience with a unique relationship to the content. The voice must feel like a real human expert in that niche - authentic, consistent, and memorable.

## Brand Consistency Rules

### DO
- Reference the brand guide BEFORE starting any design
- Use the exact color codes specified
- Maintain the voice persona throughout
- Test designs against brand checklist
- Use site-specific imagery styles

### DON'T
- Mix elements between sites
- Use generic stock photos
- Default to generic fonts
- Deviate from voice without reason
- Assume what "looks good" fits the brand

## Voice Calibration Test

Before publishing content, ask:
1. Would this persona say this exactly this way?
2. Does the tone match the audience's expectations?
3. Are technical terms appropriate for this site's readers?
4. Does it feel human-written or AI-generated?
5. Would a competitor's content feel different?

If any answer is wrong, revise.
