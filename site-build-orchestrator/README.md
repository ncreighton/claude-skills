# Site Build Orchestrator v2.0

**The Complete Empire Building System**

Build, launch, maintain, and scale any site in your publishing empire with zero guesswork, zero missed steps, and maximum efficiency.

## What's New in v2.0

- **12-Phase Build Process** (expanded from 10)
- **Site-Type Templates** - Specialized guides for affiliate, authority, lifestyle, and news sites
- **n8n Automation Integration** - Complete workflow templates for hands-off publishing
- **Troubleshooting & Emergency Recovery** - Comprehensive problem resolution guides
- **Post-Launch Maintenance System** - Weekly, monthly, quarterly maintenance frameworks
- **Content Strategy System** - Keyword research, pillar-cluster planning, content calendars
- **Validation & QA Automation** - Automated quality checks against DNA specifications
- **Batch Generator** - Generate build packages for all 16 sites at once

## Quick Start

```
Generate complete build package for [SiteName]:
1. Read DNA from creative-command-center
2. Generate CLAUDE.md
3. Generate BUILD-GUIDE.md (site-type specific)
4. Generate all checklists
5. Save to outputs folder
```

## Integration

Works with:
- **Creative Command Center** - DNA files (design specs)
- **CLAUDE.md Generator** - Design tokens
- **n8n** - Automation workflows
- **ZimmWriter** - Content generation
- **WordPress MCP** - Site execution

## The 12-Phase Build Process

| Phase | Name | Purpose |
|-------|------|---------|
| 0 | Preflight | Verify requirements |
| 1 | Foundation | Theme, plugins |
| 2 | Design System | Tokens, colors |
| 3 | Global Elements | Header, footer |
| 4 | Homepage | Hero, sections |
| 5 | Templates | Category, post |
| 6 | Core Pages | About, legal |
| 7 | SEO Setup | RankMath, schema |
| 8 | Performance | Caching, CWV |
| 9 | Content | Pillars, linking |
| 10 | Integration | Analytics, email |
| 11 | QA & Testing | Full testing |
| 12 | Launch | Go live |
