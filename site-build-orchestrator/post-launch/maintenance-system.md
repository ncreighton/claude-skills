## Site Lifecycle Decisions

### When to Double Down
```
Indicators:
- Growing organic traffic
- Increasing revenue
- Competitive advantage
- Passionate about topic
- Scalable content opportunities

Actions:
- Increase content investment
- Build team/contractors
- Develop products
- Pursue partnerships
```

### When to Maintain
```
Indicators:
- Stable traffic/revenue
- Mature market
- Limited growth potential
- Still profitable

Actions:
- Minimize investment
- Focus on maintenance
- Extract value
- Consider monetization changes
```

### When to Sunset
```
Indicators:
- Declining traffic
- Negative ROI
- No passion for topic
- Better opportunities elsewhere
- Algorithm penalty

Options:
- Sell the site
- Redirect to another property
- Archive content
- Let domain expire
```
